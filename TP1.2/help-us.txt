*** HELP ***
------------------------------------------------------------------------------------------------
Program's arguments
------------------------------------------------------------------------------------------------
Arg.	Option	Description
------------------------------------------------------------------------------------------------
-h		Shows help
------------------------------------------------------------------------------------------------
-m	N	Simpletron has a memory of N size. By omission, N equals 50.
------------------------------------------------------------------------------------------------
-i	file	The program will be read from the file indicated. By omission, read from stdin.
------------------------------------------------------------------------------------------------
-if	bin	Entry file is of type Bin.
	txt	Entry file is of type text.
------------------------------------------------------------------------------------------------
-o	file	Final information will be dumped into the file indicated. By omission, stout.
------------------------------------------------------------------------------------------------
-of	bin	The dump will be in binary form.
	txt	The dump will be in text form.
------------------------------------------------------------------------------------------------
